A real-time chat application built using socket programming, enabling multiple clients to communicate with a central server over TCP. This project demonstrates core networking concepts like client-server architecture, message broadcasting, and concurrent connection handling.

🚀 Features
Real-time messaging between multiple users
Client-server architecture using TCP sockets
Multi-client handling (threads)
Broadcast messages to all connected clients
Simple and lightweight console-based interface
Graceful client connect/disconnect handling

🛠️ Tech Stack
Language: Java
Networking: Socket, ServerSocket
Multithreading: Thread / ExecutorService
